# Customer Segmentation using K-Means Clustering

## Project Overview
This project focuses on customer segmentation using the K-Means clustering algorithm. The goal is to group customers into different clusters based on their purchasing behavior to help businesses identify target marketing groups.

## Dataset
The dataset contains customer information used for segmentation analysis.

Features used:
- Customer ID
- Annual Income
- Spending Score

## Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Methodology
The project follows these steps:

1. Data loading and preprocessing
2. Exploratory Data Analysis (EDA)
3. Determining optimal clusters using the Elbow Method
4. Applying K-Means clustering algorithm
5. Visualizing the clusters

## Elbow Method
The Elbow Method is used to determine the optimal number of clusters by plotting the Within-Cluster Sum of Squares (WCSS) against the number of clusters.

## Cluster Visualization
Customers are grouped into clusters and visualized using scatter plots to understand different spending patterns.

## Results
The model successfully segments customers into distinct groups based on income and spending score, helping businesses design targeted marketing strategies.
